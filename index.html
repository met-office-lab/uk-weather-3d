Uk-weather-3d

Experimental UK weather forecast viewer using threejs and modified WCS server

Download this project as a .zip file Download this project as a tar.gz file

Try it now.

This work in progress is an experiment in web based 3D display of high spatial resolution (approx 1km at highest res) UK weather forecasts.

The links here can be used to see some examples (if everything is working well). Note this IS NOT IN ANY WAY AN OPERATIONAL, OR EVEN USABLE, SERVICE. IT'S AN EXPERIMENT.
